## [1.0.1] — 2026-05-10

### 수정 (Fixed)
- `toIng()` 이중모음 자음 겹침 버그
  - eat → ~~eatting~~ → **eating** ✅
  - read → ~~readding~~ → **reading** ✅
  - look → ~~lookking~~ → **looking** ✅
  - 원인: `/[aeiou][자음]$/` 패턴이 이중모음(ea, oo 등)에도 겹침 적용
  - 수정: `/[^aeiou][aeiou][자음]$/` 로 단일모음(CVC) 패턴만 겹침 적용
- 전수 테스트 54/54 통과 확인

## [1.0.0] — 2025-05-09

### 추가 (Added)
- 🎤 **쉐도잉 모드** (5번째 탭)
  - MediaRecorder API로 실제 오디오 캡처
  - 모범 TTS → 내 녹음 → 비교 청취 3단계 흐름
  - 실시간 파형 시각화 (상태별 색상 변화)
  - 속도 조절 슬라이더 (0.5x ~ 1.2x, 기본 0.8x)
  - 퀴즈 현재 문장과 자동 동기화
- 📋 **학습목표 관리** (DB 관리 → 학습목표 탭)
  - 목표 생성/삭제 (컬러 자동 배정)
  - 문장을 다른 목표로 이동
  - 📢 연속 읽기: 목표 내 전체 TTS 자동 재생
  - 🎤 목표 → 쉐도잉 모드 바로 진입
- 🔴🟠🟡🟢 **오답 횟수 시각화 배지** (DB 편집 탭)
  - 0회: 🟢 완벽 / 1~4회: 🟢 N회
  - 5~9회: 🟡 N회 / 10~14회: 🟠 N회 / 15회↑: 🔴 N회
- 🏠 **환영 화면** (재방문 시)
  - 이어하기 (마지막 위치 자동 복원)
  - 핵심 복습하기 (SRS due 문장 바로 시작)
  - 처음부터 다시 하기
  - 완료 문장 수 / 오늘 복습 / 정답률 / 스트릭 요약 표시
- ⚙️ **설정 패널** (⚙️ 버튼)
  - 한국어 문장 보기/숨기기 토글
  - 한국어 자동 음성 토글
  - 정답 자동 음성 토글
  - 힌트 보기 토글
  - 정답 보기 = 오답 처리 토글
- 💾 **학습 위치 자동 저장** (문제 이동마다 localStorage 저장)
- ⌨️ **타이핑 모드** (STT 미지원 환경 대비, Enter로 채점)
- 👆 **스와이프 제스처** (좌/우 60px 이상 → 이전/다음 문제)
- ⌨️ **키보드 단축키** (← → 이동 / Space 마이크 / Enter 정답 보기)
- 🎨 **새 앱 아이콘** (민트 V1 — 말풍선 + E 레터 디자인)

### 수정 (Fixed)
- `switchMode('quiz')` 전환 시 render() 미호출 버그
- OPIc / 문장늘리기 탭 재방문 시 초기화 버그 (`_initialized` 플래그)
- `showAns()` SRS 기록 누락 버그 (정답 보기 눌러도 학습 기록 안됨)
- 내장 DB 편집 시 무경고 커스텀 전환 (confirm 다이얼로그 추가)
- GoalManager 학습목표 탭 미리보기 미갱신 버그
- 연속 읽기 타이밍 불안정 (시간추정 → onend 이벤트 기반으로 수정)
- iOS Safari 홈 화면 추가 안내 문구 수정
  - Before: `Safari 하단 공유 버튼 탭`
  - After: `우측 하단 ··· 아이콘 → 공유 → 홈 화면에 추가`

### 아키텍처 (Architecture)
- `ShadowMode` 객체 추가 (파형·녹음·플레이리스트 관리)
- `GoalManager` 객체 추가 (학습목표 CRUD + localStorage)
- `wrongBadge()` 함수 추가 (오답 횟수 → 이모지 배지 변환)
- 모드바 4탭 → 5탭 확장 (영작/변형/쉐도잉/늘리기/OPIc)
- DB 관리 3탭 → 4탭 (DB선택/학습목표/업로드/편집)
